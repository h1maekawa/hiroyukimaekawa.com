Features

必要な機能だけを、
シンプルに。

01

マルチサイト自動スクレイピング

食べログ・ホットペッパー・Google Mapsの3大プラットフォームに対応。サイトごとのUI差異を吸収し、統一フォーマットでデータを取得します。

02

電話番号の精密抽出

各サイト固有のDOM構造を解析し、電話番号を正確に抽出。形式の違い（ハイフン有無・全角半角）も自動で正規化します。

03

ワンクリックCSV出力

取得したデータは即座にCSV形式でダウンロード。Excel・Googleスプレッドシートで開くだけで営業リストとして活用可能です。

04

重複排除・データクレンジング

同じ店舗が複数サイトに掲載されていても自動で検出・排除。常にクリーンなデータセットを維持します。

How It Works

3ステップで完了。
複雑な設定は不要です。

01

拡張機能をインストール

GitHubからダウンロードし、Chromeの拡張機能画面からインストール。デベロッパーモードで「パッケージ化されていない拡張機能を読み込む」を選択するだけです。

02

対応サイトで検索

食べログ、ホットペッパー、Google Mapsのいずれかで通常通り検索。「渋谷 居酒屋」「新宿 イタリアン」など、営業したいエリア×ジャンルで検索します。

03

ワンクリックで抽出 → CSV完成

拡張機能アイコンをクリックし「リスト取得開始」を押すだけ。検索結果の全店舗情報が自動抽出され、CSVファイルとしてダウンロードされます。完了時にはデスクトップ通知でお知らせ。

Trust & Tech

オープンソースだから、
安心して使えます。

JavaScript + Chrome APIs

Chrome Extension Manifest V3準拠。Content Scripts と Background Service Worker による安定した動作を実現しています。

GitHubで全コード公開

ソースコードは100%公開。動作内容を誰でも確認でき、セキュリティリスクへの不安を解消します。MIT License。

安全なローカル動作

取得したデータは外部サーバーに送信されません。すべてブラウザ内で完結し、CSVとしてローカルに保存されます。
