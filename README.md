# HIROYUKI MAEKAWA Portfolio

静的サイト（HTML / CSS / JavaScript）です。Viteは開発サーバーとビルドに利用しています。

## ディレクトリ構成

- `index.html`: トップページ
- `works.html`: 制作実績一覧ページ
- `assets/css/styles.css`: スタイル
- `assets/js/main.js`: トップページ用スクリプト
- `assets/js/works.js`: 実績一覧ページ用スクリプト
- `assets/img/`: 画像ファイル

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
